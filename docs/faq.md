# Preguntas frecuentes

[← Volver al inicio](../README.md)

### El video no se ve / sale negro
- Espera unos segundos: el video puede tardar en conectar.
- Las cámaras **LPR** se ven casi negras de noche (obturador rápido + infrarrojo
  para leer placas). Es normal.
- Si **ninguna** cámara reproduce, puede ser un problema de red entre tu equipo y
  el sistema; avisa al administrador.

### Un marcador está en rojo pero la cámara funciona
El color refleja el último sondeo de estado. Algunas cámaras solo se conectan
"bajo demanda"; aun así, al abrir el panel el video normalmente sí reproduce.

### ¿Por qué un marcador es ámbar?
Porque en ese sitio **algunas** cámaras están en línea y **otras** fuera. Haz
clic para ver, en las fichas, cuál está caída.

### La barra de disponibilidad está gris
Aún no hay suficiente historial (el sistema registra el estado cada minuto). Con
el paso de las horas y días la barra se llena.

### No veo Grupo / IP / Stream de la cámara
Esos datos son solo para el rol **administrador**. Como **visor** solo se muestra
el **Servidor**.

### ¿Cómo atiendo una alarma SOS?
Abre la campana 🔔, ubica la alarma, usa **Reconocer** mientras la atiendes y
**Resolver** al cerrarla (puedes dejar una nota). Todo queda registrado con tu
usuario.

### El asistente dice que no hay resultados
Significa que no encontró datos para esa pregunta en el periodo. Prueba a ampliar
las fechas o cambiar los términos.

### Olvidé mi contraseña
Un administrador la restablece en **Administración → Usuarios**.
